Windows Compilation instructions:
------------------------------------
Install the following software IN THE ORDER LISTED into their default locations:

- mingw32               - http://sourceforge.net/projects/mingw/files/Installer/mingw-get-inst/
     *** Install at least the gcc stuff from mingw32 ***

Once you've installed these, add the following directories to your system path:
C:\MinGW\bin\

If you did the above, you should be able to double click build.bat in this folder to compile.
